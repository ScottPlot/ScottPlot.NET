<div class='mt-3 fs-4'><strong>Plot Types</strong></div>
<ul>
<li><a href='/cookbook/5/Annotation'>Annotation</a> - Annotations are always-visible text labels positioned over the data area.</li>
<li><a href='/cookbook/5/ArrowCoordinated'>Arrow</a> - Arrows point to a location in coordinate space.</li>
<li><a href='/cookbook/5/AxisLines'>Axis Lines</a> - Axis lines indicate a position on an axis.</li>
<li><a href='/cookbook/5/AxisSpans'>Axis Spans</a> - Axis spans indicate a range of an axis.</li>
<li><a href='/cookbook/5/Bar'>Bar Plot</a> - Bar plots represent values as horizontal or vertical rectangles</li>
<li><a href='/cookbook/5/Box'>Box Plot</a> - Box plots show a distribution at a glance</li>
<li><a href='/cookbook/5/Bracket'>Bracket</a> - Brackets annotate a range along a line in coordinate space</li>
<li><a href='/cookbook/5/Callout'>Callout</a> - Callouts display a label and are connected with an arrow that marks a point on the plot.</li>
<li><a href='/cookbook/5/Contour'>Contour Plot</a> - A contour plot is a graphical representation that shows the three-dimensional surface of a function on a two-dimensional plane by connecting points of equal value with contour lines</li>
<li><a href='/cookbook/5/Coxcomb'>Coxcomb Plot</a> - A Coxcomb chart is a pie graph where the angle of slices is constant but the radii are not.</li>
<li><a href='/cookbook/5/Crosshair'>Crosshair</a> - A Crosshair combines a horizontal axis line and vertical axis line to mark a location in coordinate space.</li>
<li><a href='/cookbook/5/Ellipse'>Ellipse</a> - Ellipses are curves with a defined center and distinct X and Y radii. A circle is an ellipse with an X radius equal to its Y radius.</li>
<li><a href='/cookbook/5/ErrorBar'>Error Bars</a> - Error Bars communicate the range of possible values for a measurement</li>
<li><a href='/cookbook/5/FillY'>FillY plot</a> - FillY plots display the vertical range between two Y values at defined X positions</li>
<li><a href='/cookbook/5/Finance'>Financial Plot</a> - Finance plots display price data binned into time ranges</li>
<li><a href='/cookbook/5/Function'>Function</a> - Function plots are a type of line plot where Y positions are defined by a function that depends on X rather than a collection of discrete data points.</li>
<li><a href='/cookbook/5/Heatmap'>Heatmap</a> - Heatmaps display values from 2D data as an image with cells of different intensities</li>
<li><a href='/cookbook/5/Images'>Images</a> - Images can be placed on plots in a variety of ways</li>
<li><a href='/cookbook/5/InteractivePlottables'>Interactive</a> - Interactive plottables interact with the mouse without requiring the user to manually wire mouse tracking.</li>
<li><a href='/cookbook/5/LinePlot'>Line Plot</a> - Line plots can be placed on the plot in coordinate space using a Start, End, and an optional LineStyle.</li>
<li><a href='/cookbook/5/LiveData'>Live Data</a> - Plottables like DataLogger and DataStreamer are designed for displaying datasets that change in real time. They have the ability to control axis limits to ensure the latest data is always in view. See the ScottPlot Demo for live example of these plot types.</li>
<li><a href='/cookbook/5/Lollipop'>Lollipop Plot</a> - A lollipop chart is a variation of a bar chart that uses a line (stem) extending from a baseline to a marker (head) to represent data points. Lollipop highlight individual data points with less visual clutter than to traditional bar charts.</li>
<li><a href='/cookbook/5/Marker'>Marker</a> - Markers can be placed on the plot in coordinate space.</li>
<li><a href='/cookbook/5/Phasor'>Phasor Plot</a> - Phasor plots display vectors on a radial axis centered at the origin</li>
<li><a href='/cookbook/5/Pie'>Pie</a> - Pie charts illustrate numerical proportions as slices of a circle.</li>
<li><a href='/cookbook/5/PolarAxis'>Polar Axis</a> - Create a polar axis and add it to the plot to display data on a circular coordinate system.</li>
<li><a href='/cookbook/5/Population'>Population Plot</a> - Population plots display collections of individual values.</li>
<li><a href='/cookbook/5/Radar'>Radar Plot</a> - Radar charts (also called a spider charts or star charts) represent multi-axis data as a 2D shape on axes arranged circularly around a center point.</li>
<li><a href='/cookbook/5/RadialGauge'>Radial gauge</a> - A radial gauge chart displays scalar data as circular gauges.</li>
<li><a href='/cookbook/5/ScaleBar'>Scale Bar</a> - Scalebars display a horizontal and/or vertical range using a line segment and may be used to convey axis scale as a minimal alternative to using axis frames, ticks, and tick labels.</li>
<li><a href='/cookbook/5/Scatter'>Scatter Plot</a> - Scatter plots display points at X/Y locations in coordinate space.</li>
<li><a href='/cookbook/5/Shapes'>Shapes</a> - Basic shapes that can be added to plots</li>
<li><a href='/cookbook/5/Signal'>Signal Plot</a> - Signal plots display evenly-spaced data</li>
<li><a href='/cookbook/5/SignalConst'>SignalConst</a> - SignalConst is a type of signal plot which contains immutable data points and occupies more memory but offers greater performance for extremely large datasets. It is rarely needed, but best use for plotting data containing millions of points.</li>
<li><a href='/cookbook/5/SignalXY'>SignalXY Plot</a> - SignalXY are a high performance plot type optimized for X/Y pairs where the X values are always ascending. For large datasets SignalXY plots are much more performant than Scatter plots (which allow unordered X points) but not as performant as Signal plots (which require fixed spacing between X points).</li>
<li><a href='/cookbook/5/SmithChart'>Smith Chart</a> - Create a Smith chart axis and add it to the plot to display impedance of RF signals using a horizontal axis indicating resistance and vertical axis indicating reactance.</li>
<li><a href='/cookbook/5/Text'>Text</a> - Text labels can be placed on the plot in coordinate space</li>
<li><a href='/cookbook/5/Tooltip'>Tooltip</a> - Tooltips are annotations that point to an X/Y coordinate on the plot.</li>
<li><a href='/cookbook/5/TriangularAxis'>Triangular Axis</a> - Create a triangular axis and add it to the plot to display data on a triangular coordinate system.</li>
<li><a href='/cookbook/5/VectorField'>Vector Field</a> - Vector fields display a collection of vectors rooted at points in coordinate space</li>
</ul>
